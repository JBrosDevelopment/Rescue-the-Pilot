Search and Rescue: Find the Pilot

Unity WebGL Player | Search and Rescue
Search and Rescue

Controls: Move - WASD and Arrow Keys Release a bogey - Space or B Point of Game: The Point of the game is to find a stranded pilot that has just crashed. You can release bogeys to help you along the way. They will warn you if the pilot or any debris has drifted near it. You have only so much time to find the pilot, so good luck! Developer Comments: This game might have a bug or two, an example would be that the exit button doesn't work. I ended up in a rush with only 15 minutes left when I submit. Unity crashed, a couple of times, so I struggled a bit. In all, I spent 8 and a half hours on this game over 24 hours. Stuff I Used: Music from patrickdearteaga.com, Sound Effects from Pixabay
